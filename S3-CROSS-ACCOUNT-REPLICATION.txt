TASK : Create S3 bucket and cross ACCOUNT replication

1.(First let's create a s3 bucket )
		1.select S3 services and CREATE BUCKET
		2.Bucket name 
		3.AWS Region 
		4.Select ACL DISABLE
		5.KEEP other option default
		6.ENABLE BUCKET VERSIONING

		7.Keep remaining setting as default
		8.create Bucket 
		9.OPEN THE CREATED BUCKET AND IN OBJECT SECTION SELECT UPLOAD 
		10.UPLOAD THE FILES OR FOLDER U WANT TO UPLOAD 
		
2.IN ANOTHER ACCOUNT CREATE THE ANOTHER BUCKET WITH STEPS AS MENTIONED ABOVE.

3.(NOW TO ENABLE BUCKET cross region replication )

		1.SELECT THE FIRST BUCEKT WE CREATED
		2.SELECT THE MANAGEMENT TAB
		3.SELECT CREATE  REPLICATION RULE 
		  -1.ENTER REPLICATION RULE NAME 
		  -2.STATUS : ENABLE 
		  -3.IN SOURCE BUCKET TAB :-
				   -CHOSE A RULE SCOPE : [APPLY TO ALL OBJECT IN BUCEKT]
		  -4.DESTINATION :-
						   -SELECT CHOSE BUCKET IN ANOTHER AWS ACCOUNT 
						   -ENTER ANOTHER ACCOUNT ID
						   -ENTER BUCEKT NAME
						   -SELECT CHANGE OBEJCT OWNERSHIP TO DESTINATION BUCEKT OWNER
						   
		  -5.IAM Role :- 
						-SELECT CHOSE FROM EXISTING IAM Roles 
						 -CHOSE  CREATE A NEW ROLE
		  -6.destination storage class :- (if you want to chose another  storage class you can select that.)
		  -7.save 
		  -8.in replication existing object dialog box :-
														SELECT YES,REPLICATE EXISTING OBJECT(ONLY IF HAVE EXISTING DATE AND YOU WANT TO REPLICATE THAT,OTHERWISE CHOSE NO OPTION)
		  -9.SUBMIT
4. YOU WILL ABLE TO SEE THE Created REPLICATION RULE:- 
                                -SELECT ROLE TAHT WE HAVE CREATE AND COPY THE POLICY 

5.GO TO THE DESTINATION BUCKET:-
        - IN PERMISSION, GO TO THE BUCEKT POLICY AND EDIT 
        -APPLY THE BUCEKT POLICY AND ENUSRE TO CHANGE THE ARN NUMBER WITH THE DESTINATION BUCKET ARN NUMBER.
        -SAVE CHANGES 
6.(TO CHECK WHEATHER REPLICATION RULE IS WORKING OR NOT )
   1.SELECT 1ST BUCKET AND IN OBJECT SECTION,SELECT UPLOAD
   2. UPLOAD ANY NEW OBJECT 
   3.THEN SELECT THE ANOTHER ACCOUNT BUCKET.
      - YOU WILL ABLE TO SEE THE SAME OBJECT THAT WE UPLOADED IN FIRST BUCKET (Some time it takes time so try to referesh.)
